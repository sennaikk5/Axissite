# Desktop Browser Android

Starter project for a fullscreen, desktop-like browser with a virtual mouse overlay.

## Included
- Kotlin + XML Android app
- fullscreen immersive mode
- desktop user-agent WebView
- URL bar
- touch/mouse hybrid overlay starter

## Build notes
This is a source project ready to import into Android Studio, AndroidIDE, or a cloud builder.
If you use a GitHub-based CI builder, you may need to add a Gradle wrapper before the first build.
